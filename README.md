# align-it-ob3


## Description

Code for align-it with openbabel3


## Requirements

openbabel3


## INSTALL

- following example is basic way.

```
git clone https://github.com/iwatobipen/align-it-ob3.git
cd align-it-ob3
mkdir biuild
cd build
cmake -DCMAKE_INSTALL_DIR=<where you want to insatall> ..
make
make install
```
